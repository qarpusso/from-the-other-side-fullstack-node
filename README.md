# Habit & Note Tracker

A simple habit tracking and markdown note-taking application built with Node.js, vanilla JavaScript, and Supabase.

## Features

- Track daily and weekly habits
- Log habit completions with optional notes
- Create and manage markdown notes
- Real-time markdown preview
- Responsive design

## Installation

```bash
npm install
```

## Running the Application

```bash
npm start
```

The application will be available at http://localhost:8000
